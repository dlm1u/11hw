# Приключение в Мистик Фоллс

В этом проекте реализован Telegram-бот, который представляет собой интерактивный текстовый квест. Пользователь ("игрок") должен пройти регистрацию и двигаться по сюжету, делая выбор в ключевых точках повествования.
